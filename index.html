计算公式与符号说明
回归系数
Y = B·X + A
B = (Σxy - (Σx)(Σy)/n) / (Σx² - (Σx)²/n)
A = ȳ - B·x̄
相关性
r = (Σxy - (Σx)(Σy)/n) / √[(Σx² - (Σx)²/n)(Σy² - (Σy)²/n)]
R² = r²
不确定度
s = √(Σe² / (n - 2))
sB = s / √(Σx² - (Σx)²/n)
sA = s·√(Σx² / (n(Σx² - (Σx)²/n)))
符号
n: 数据点数；e: 残差；ȳ, x̄: 样本均值；
Σxy, Σx², Σy² 分别为乘积与平方和。
